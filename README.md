# SQL Sales Analysis

This project demonstrates a simple data analysis workflow using Python, SQL and SQLite.

The pipeline loads sales data from a CSV file, stores it in a SQLite database and runs SQL queries to analyze the data.

## Technologies used

- Python
- Pandas
- SQLite
- SQL

## Project Structure

sql-sales-analysis

data/ → raw dataset  
queries/ → SQL queries  
python/ → Python analysis script  

## Example Analysis

The script calculates:

• Revenue by region  
• Average revenue by category  
• Monthly revenue trends
